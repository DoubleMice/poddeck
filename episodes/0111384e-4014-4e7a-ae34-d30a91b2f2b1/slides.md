# Etched：把 Transformer "刻进"芯片

<div class="mt-3 grid grid-cols-3 gap-3 text-sm leading-relaxed">

<div class="col-span-1 p-3 rounded bg-blue-50 border-l-4 border-blue-500">

### 🎯 核心赌注
两年前，Etched 押注 Transformer 架构不会过时。今天这个赌注已被验证——模型越来越大，Transformer 依然是主干

</div>

<div class="col-span-1 p-3 rounded bg-green-50 border-l-4 border-green-500">

### 🔥 低压推理
传统 GPU 只能跑到理论算力的 ~50%，否则会熔毁。Etched 通过**降电压 + TSMC 联合设计晶体管**，大幅提升每瓦算力

</div>

<div class="col-span-1 p-3 rounded bg-yellow-50 border-l-4 border-yellow-500">

### 🧠 集群级内存
将整个扩展域（scale-up domain）的 HBM + SRAM 作为**单一一致性内存池**，减少芯片间通信延迟，提升 token 生成速度

</div>

</div>

<div class="mt-2 p-3 bg-gray-50 border-l-4 border-gray-400 rounded text-xs leading-relaxed">
<strong>Gavin Uberti, CEO</strong>：芯片行业是精英统治——Toyota 和 Ferrari 不是一回事。速度差距是 2 倍以上，如果你能更快，就能拿到巨大溢价。我们的护城河不是软件生态，而是持续的技术创新和全栈整合。
</div>

# 推理硬件的两个瓶颈

<div class="text-sm leading-relaxed space-y-3">

<div class="p-3 rounded bg-red-50 border-l-4 border-red-500">

### Prefill（计算密集）
- 需要更多 **FLOPS**（每秒操作数）
- GPU 瓶颈在**热管理**——理论算力被发热限制在 50%
- Etched 方案：**低压推理** → 功耗大幅降低 → 可在芯片上堆更多算力

</div>

<div class="p-3 rounded bg-blue-50 border-l-4 border-blue-500">

### Decode（带宽密集）
- 需要更多 **内存带宽**
- 瓶颈不仅在单芯片，更在**芯片间通信**
- Etched 方案：**集群级内存** → 整个扩展域的 HBM 作为统一池 → 大幅降低通信延迟

</div>

<div class="p-3 bg-gray-50 border-l-4 border-gray-400 rounded text-xs">
<strong>作者概括</strong>：Gavin 的论点本质是 "scaling laws 适用于芯片制造"——市场需求越大（万亿参数模型、千万亿 token 需求），越值得投资规模化的专用硬件。
</div>

# Monogram：AI 交互的下一代范式

<div class="grid grid-cols-2 gap-4 mt-4 text-sm leading-relaxed">

<div>

### 从文本到界面

<div class="p-3 rounded bg-blue-50 border-l-4 border-blue-500 mt-2">

**传统 AI**：打字 → 收到文字回复

**Monogram**：说话 → 1.5 秒内看到**完整的可视化界面**——信息架构、卡片、可交互元素

</div>

### 为什么是语音优先

<div class="p-3 rounded bg-green-50 border-l-4 border-green-500 mt-3">

- 说话比打字**快 4-5 倍**（每秒信息量）
- 但听比读**慢 4 倍**
- 最优解：**语音输入 + 视觉输出**

</div>

</div>

<div>

### Eren Bali 的连续创业

<div class="p-3 rounded bg-yellow-50 border-l-4 border-yellow-500 mt-2">

<strong>Udemy</strong>（在线教育，已上市）→ <strong>Carbon Health</strong>（科技医疗）→ <strong>Monogram</strong>（AI 应用）

# 数据中心液冷：AI 时代的"血液监测"

<div class="grid grid-cols-2 gap-4 mt-4 text-sm leading-relaxed">

<div>

### Omen AI：来自采矿业的启发

<div class="p-3 rounded bg-blue-50 border-l-4 border-blue-500 mt-2">

创始人 Zach Laberge 最初在**大型矿用卡车**上做油液监测，后来发现数据中心液冷系统面对同样的问题

<div class="text-xs opacity-70 mt-1">公司名 "Omen" 意为"预兆"——你希望预测的是好兆头（机器不坏），而非坏兆头</div>

</div>

### 液冷系统如何退化

<div class="p-3 rounded bg-red-50 border-l-4 border-red-500 mt-3">

- 泵和密封件磨损 → **铜、铬、硅**渗入冷却液
- 高温 + 高浓度乙二醇 → 流体化学分解 → **生物生长**
- 污染物沉积在芯片表面和管道内 → 腐蚀和堵塞

</div>

</div>

<div>

### 为什么需要实时监测

<div class="p-3 rounded bg-yellow-50 border-l-4 border-yellow-500 mt-2">

**现状**：从数据中心取样 → 寄到实验室 → 等几周拿到报告 → 结论通常是"要么全部冲洗"

</div>

<div class="p-3 rounded bg-green-50 border-l-4 border-green-500 mt-3">

**冲洗一个机架 = 4-5 小时停机 × 1,000 GPU = 5,000 GPU 小时浪费**

</div>

<div class="p-3 rounded bg-blue-50 border-l-4 border-blue-500 mt-3">

**Omen 方案**：直接接入机架歧管，每天实时监测 → 只需调整化学配方（生物冲击 / 乙二醇比例）→ 避免大规模停机冲洗

</div>

<div class="p-3 bg-gray-50 border-l-4 border-gray-400 rounded mt-3 text-xs">
<strong>$3100 万 A 轮</strong>，投资人包括 Cheryl Sandberg（Nova Ventures）、CRV，以及 CoreWeave 和 Supermicro 高管。按 Gavin 的逻辑——推理需求越大，数据中心越多，Omen 的市场越大。
</div>

# 硬科技投资的哲学：速度胜于深度

<div class="mt-4 text-sm leading-relaxed">

<div class="grid grid-cols-2 gap-4">

<div>

<div class="space-y-2">

<div class="p-3 rounded bg-blue-50 border-l-4 border-blue-500">

<strong>不招博士</strong> — "如果要商业化自己的博士论文，你手里只有一把锤子，到处找钉子。这种模式几乎从不成功。博士生也被训练成'处处加注脚、什么可能出错'——完全相反的心态。"

</div>

<div class="p-3 rounded bg-green-50 border-l-4 border-green-500">

<strong>迭代速度决定一切</strong> — "我们筛选时如果在 7 天内看不到进展，就不会在 1 个月、1 季度、1 年后看到。对有些人来说，一周等于别人的一个月。"

</div>

<div class="p-3 rounded bg-yellow-50 border-l-4 border-yellow-500">

<strong>技术本身没有价值</strong> — "价值在于解决问题、创造世界上的实际产出。把货架上的技术以新方式打包，比'我有一个很酷的博士成果怎么卖'有趣得多。"

</div>

### Shinkei 案例：从授权到品牌的 pivot

<div class="p-3 rounded bg-red-50 border-l-4 border-red-500 mt-2">

<div class="p-3 rounded bg-green-50 border-l-4 border-green-500 mt-2">

<div class="p-3 rounded bg-blue-50 border-l-4 border-blue-500 mt-2">

结果：完全垂直整合——**用自己的机器人杀自己的鱼，卖自己的品牌**。比单纯授权机器人赚得多得多。

</div>

<div class="p-3 rounded bg-orange-50 border-l-4 border-orange-500 mt-2">

<strong>Grant Gregory</strong>（从 a16z 加入 Cantos）："最贵的不是硬件，是人。StarShield 和 Starship 的成本在宏观上不高——真正贵的是投入的工时。所以越早测试、越早破坏、越早重构，越好。"

<strong>例</strong>：Radiant 的核反应堆最初瞄准军用，现在数据中心的 behind-the-meter 应用反而成为主要增长驱动力。

</div>

<div class="p-3 rounded bg-orange-50 border-l-4 border-orange-500 mt-3">

<strong>AI 不是取代硬科技，而是喂养硬科技</strong>。但 Cantos 提醒：创始人不能说"我们要做 AI 版的东西"，而是理解"AI 需求如何使我的技术更有价值"。

</div>

# 消费者愿望与创业机会的交汇

### 看似"蠢"的问题往往是最大的机会

<div class="p-3 rounded bg-blue-50 border-l-4 border-blue-500">

<strong>降噪干墙</strong> — "声音是波，难道不能屏蔽？" → 超轻隔音膜用于现有建筑 → 初创公司机会

</div>

<div class="p-3 rounded bg-yellow-50 border-l-4 border-yellow-500">

<strong>智能防风玻璃</strong> — 自动在阳光或反射光照射的位置变色 → 电致变色玻璃已有部分实现

<strong>可升降洗碗机</strong> — "让高个子不用弯腰" → 已存在但不够高！6'8" 的人仍然弯腰

<div class="p-3 rounded bg-yellow-50 border-l-4 border-yellow-500 mt-3">

# 芯片会变成汽车一样的商品吗？

<div class="grid grid-cols-2 gap-4 mt-4 text-sm leading-relaxed">

<div>

### 🚗 "芯片 = 汽车" 论点

<div class="p-3 rounded bg-blue-50 border-l-4 border-blue-500 mt-2">

- 我可以明天换一辆新车，连上蓝牙，它就像任何商品一样工作
- Agentic coding 降低了为不同芯片写高效软件的门槛
- 高回报激励厂商支持多种 ASIC——生态锁定在瓦解

</div>

</div>

<div>

### 🏎️ "芯片 = Ferrari" 论点（Gavin Uberti）

<div class="p-3 rounded bg-red-50 border-l-4 border-red-500 mt-2">

- Toyota 和 Ferrari 不是一回事——速度差 2 倍
- 芯片行业是精英统治：如果你能更快，就能拿到巨大溢价
- 但光快不够——必须**在可负担的价格上**做到
- 护城河不是一个软件生态，而是**持续的技术创新**

</div>

</div>

</div>

<div class="mt-3 p-3 bg-gray-50 border-l-4 border-gray-400 rounded text-xs leading-relaxed">
<strong>作者概括</strong>：这个辩论的核心是——AI 推理市场是否足够大、足够差异化，使得专用硬件（ASIC）比通用硬件（GPU）有持续的竞争优势？Etched 赌的是"是的"。
</div>

# Oasis Devices：智能戒指替代键盘

<div class="grid grid-cols-2 gap-4 mt-4 text-sm leading-relaxed">

<div>

### 产品定位

<div class="p-3 rounded bg-blue-50 border-l-4 border-blue-500 mt-2">

<strong>戒指挥近嘴唇 → 低声口述 → 高保真转录</strong>。比 AirPods 更靠近声源，能在办公室、咖啡厅等公共场所使用——这些地方你不可能大声说话。

</div>

<div class="p-3 rounded bg-green-50 border-l-4 border-green-500 mt-3">

戒指还具备**触控板功能**——导航光标、编辑文本、与 AI 交互，完全不需要碰键盘。已与 WhisperFlow 集成。

</div>

<div class="p-3 rounded bg-yellow-50 border-l-4 border-yellow-500 mt-3">

创始人 Ricky Rosa 的策略：**"你只能要求用户迈出一步"**——不能让他们一夜之间改变全部工作流程。先解决输入问题，输出（智能眼镜等）自然会跟进。

</div>

</div>

<div>

### 硬件创业的挑战

<div class="p-3 rounded bg-red-50 border-l-4 border-red-500 mt-2">

<strong>消费者电子产品的留存率几乎为零</strong> — Apple Vision Pro 可能有 99% 流失率，大多数 VR 头显也类似

</div>

<div class="p-3 rounded bg-purple-50 border-l-4 border-purple-500 mt-3">

<strong>Oasis 的应对</strong>：
- 利用已验证的形态（戒指已有 Oura 等证明）<br/>
- 利用已验证的需求（WhisperFlow 用户疯了）<br/>
- 利用已验证的平台（电脑）而非要求用户换设备<br/>
- 5 人团队，全球分布（澳大利亚、阿布扎比、中国）

</div>

<div class="p-3 rounded bg-orange-50 border-l-4 border-orange-500 mt-3">

今日开启预购，从两周后到圣诞节分批发货。

</div>

<div class="p-3 rounded bg-blue-50 border-l-4 border-blue-500">
