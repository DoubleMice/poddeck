# Blue Origin：一次灾难性的静态点火

<div class="grid grid-cols-2 gap-4 mt-4">

<div>

**发生了什么**

Blue Origin 的 New Glenn 火箭在 LC-36 发射台进行 NG-4 任务前的静态点火测试时发生爆炸。视频显示巨大的蘑菇云升腾，目击者形容"像核弹爆炸一样"。

**关键事实**：
- New Glenn 此前 3 次发射仅 1 次完全成功
- 第三次测试中火箭正确升空但将 AST 卫星部署到了错误轨道
- 本次事故无人受伤——这被广泛视为"奇迹"
- AST SpaceMobile 股价次日下跌 16%

</div>

<div>

<div class="p-4 rounded bg-red-50 border-l-4 border-red-500 text-sm leading-relaxed">

**Jeff Bezos 的回应**：

"Very rough day. We'll rebuild whatever needs rebuilding and get back to flying. It's worth it."

**Elon Musk 的回应**：

"most unfortunate rockets are hard"

</div>

<div class="p-4 rounded bg-blue-50 border-l-4 border-blue-500 text-sm leading-relaxed mt-3">

**历史背景**：2006-2008 年，Elon Musk 的 Falcon 1 连续三次失败后才成功发射。失败是航天探索的常态——但亲眼目睹依然令人震撼。

# 航天竞争：Blue Origin vs SpaceX

<div class="grid grid-cols-2 gap-4 mt-4 text-sm leading-relaxed">

<div>

**Blue Origin 的困境**

- 比 SpaceX 更早成立，却远远落后

# New Glenn 爆炸：地缘与技术含义

<div class="grid grid-cols-2 gap-4 mt-4">

<div>

**为什么这很重要**

美国拥有多个重型发射能力提供商对整个西方航天生态至关重要。New Glenn 此前成功入轨并回收，证明了美国"第二好的火箭公司仍然领先全球其他对手"。

**结构性挑战**：

- 发射台虽未完全摧毁，但每一颗螺丝都需要重新检查
- 即使是 SpaceX 也曾经历 Starbase 的静态点火事故
- 几周后 Starship 仍然成功发射——表明快速恢复是可能的

"我认为 Blue Origin 会回来的。Elon 发来了鼓励的话，很多 SpaceX 支持者也表示支持，因为美国拥有多个重型发射能力提供商对所有人都是好事——即使是 SpaceX Maxi。"

**作者概括**: 航天不是软件工程——它是真正的"火箭科学"。爆炸虽惨烈，但对整个行业的长期信心不应动摇。

# Token Maxing：当排行榜变成目标

<div class="grid grid-cols-2 gap-4 mt-4 text-sm leading-relaxed">

<div>

**Token Maxing 现象**

企业内部出现了"token 排行榜"文化——员工让 AI 代理通宵运行非生产性任务，只是为了在消耗榜上排名靠前。

**已知案例**：
- **Meta** 内部测试了 token 排行榜，随后据说下线
- **Anthropic** 也曾有类似排行榜
- **Uber** 花光了年度 token 预算，COO 公开表示"正在努力理解财务影响和 ROI"
- 有员工用大模型查询天气预报——仅仅因为"方便"

</div>

<div>

<div class="p-4 rounded bg-orange-50 border-l-4 border-orange-500 text-sm leading-relaxed">

**节目金句**：

"Fear not the man who has vibe coded 50 apps. I fear the man who has vibe coded the same app 50 times."

— TBPN 节目讨论 vibe coding 效率悖论

</div>

<div class="p-4 rounded bg-blue-50 border-l-4 border-blue-500 text-sm leading-relaxed mt-3">

**预算悖论**: 如果你给一个营销人员 $100 万的品牌预算，他只花了 $10 万，你会问"你怎么没找到好用途？"但当员工获得 $1000-$10000 的 token 预算时，如果不用完，会被认为"没有充分利用所分配的资源"。

</div>

</div>

# Token 经济的双重法则

<div class="text-sm leading-relaxed">

**Jevons 悖论**: 当某种资源的使用效率提高时，人们往往会使用更多，而非更少。

AI token 成本持续下降 → 更多人用 → 更多场景 → 总消耗量反而上升。

**Goodhart 定律**: 当一个指标成为目标时，它就不再是一个好指标。

Token 排行榜 → 员工追求 token 量 → 非生产性消耗 → ROI 下降。

**两个法则同时成立**。Brad Gerstner 称之为"Kugen 悖论"——管理高效组织的核心挑战。

</div>

# Token 支出的看多逻辑

<div class="grid grid-cols-2 gap-4 mt-4 text-sm leading-relaxed">

<div>

**为什么短期 ROI 波动可以接受**

1. **成本递减效应**: 今天 $5 亿/月的 token 支出，明年同样的产出可能只需 1/10 的成本。硬件折旧 + 新算力上线 + 蒸馏模型 = 自然的通缩。

2. **组织转型价值**: 即使短期 ROI 不佳，让全体员工成为"AI native"工具有巨大的长期价值。一旦完成组织适配，未来每一个新用例的边际成本将大幅下降。

3. **开源与蒸馏**: 90% 的价值可能只需要 10% 的成本。企业不需要每次都调用最贵的模型。

</div>

<div>

<div class="p-4 rounded bg-green-50 border-l-4 border-green-500 text-sm leading-relaxed">

**Brad Gerstner 的调研数据**：

对 300 家企业调研显示：
- **正在优化支出的企业**仍预计 token 消耗量在未来 12 个月增长 50%+
- **计划优化支出的企业**预计增长 90%+
- 代码辅助的渗透率"刚刚起步"
- 知识工作的 AI 渗透率"几乎为零"

</div>

<div class="p-4 rounded bg-blue-50 border-l-4 border-blue-500 text-sm leading-relaxed mt-3">

**结论**: "当然人们会优化。但我们处于采用曲线的极早期。企业甚至还没真正开始用 AI 做知识工作。"

</div>

# 企业 AI ROI：董事会的新难题

<div class="text-sm leading-relaxed mt-4">

**华尔街日报的观察**: "Corporate America is starting to ration AI as costs skyrocket. Executives are scrambling to track returns on AI investments as the bill for massive computing needs come due."

<div class="grid grid-cols-3 gap-3 mt-4">

<div class="p-3 rounded bg-red-50 border-l-4 border-red-500">

**短期风险**

- 下一轮财报电话会，分析师将追问：收入是否以至少等同于增量 AI 支出的幅度增长？
- 如果用裁员节省的成本直接置换为 AI 支出，成本结构没有变化——利润结构有变化吗？
- 管理层需要在 OpEx 中展示 AI 支出的实质性影响

</div>

<div class="p-3 rounded bg-yellow-50 border-l-4 border-yellow-500">

**中间地带**

- 如果这些公司在 AI 公司中有投资，投资 mark-to-market 的收益可以抵消部分 AI 支出
- 但分析师会区分"真实收入增长"和"投资账面收益"
- Spencer Raskoff (Match Group) 表示将在未来一年系统性地追踪 AI 投资的 ROI，"就像数字营销支出一样"

</div>

<div class="p-3 rounded bg-green-50 border-l-4 border-green-500">

**长期乐观**

- 如果相信 AI 是真正的生产力工具，成本随时间递减
- 短期"块状"ROI 是合理的：组织需要花钱学习和适配
- 董事会不太可能因为几个月超预算就与管理层翻脸
- **核心问题**：这些支出是否指向了真正高杠杆的问题？还是只是在清理本来就不该做的积压事项？

</div>

# John Gruber：Ferrari Luche — 贵的不是问题，丑的才是

<div class="grid grid-cols-2 gap-4 mt-4 text-sm leading-relaxed">

<div>

**Gruber 的"逆向"观点**

Ferrari 发布了其首款电动车 Luche——一台 $65 万的五座跨界车。Gruber 和 Ben Thompson 在 Dithering 播客上给出了尖锐评价。

"如果你把车标摘掉，开过街角，问路人这是什么品牌——Ferrari 可能会排在非常非常后面。你可能要列举几十个品牌才能猜到。"

核心问题不是"电动车"，而是"它不像一台 Ferrari"。Ferrari 其实几年前已经出过燃油跨界车，所以这并非品类问题，而是设计语言问题。

</div>

<div>

<div class="p-4 rounded bg-red-50 border-l-4 border-red-500 text-sm leading-relaxed">

**折旧风险**：节目主持人 Jordy 指出，这台 $65 万的车将在几年内贬值到 $30 万甚至更低——"我不认识任何一个愿意吞下这种折旧的朋友。"

**Gruber 的评论**："我无法看到 $65 万的价值。我不是那种能被 OpenAI 收购的人。"

</div>

<div class="p-4 rounded bg-blue-50 border-l-4 border-blue-500 text-sm leading-relaxed mt-3">

**与 Apple 汽车计划的类比**：Apple 的 Titan 项目传闻最终造出的原型车成本在 $20 万以上，如果加上 Apple 的利润率——它不会是大众产品，就像 Apple Watch 如果只有 $2 万的纯金版一样。

</div>

# "设计不是它看起来怎样<br>而是它怎样工作"

<div class="text-sm leading-relaxed mt-4">

John Gruber 回溯了 Steve Jobs 关于设计的经典名言。当被问及 Apple 是否在 AI、VR、汽车等领域依然保持着"我们比消费者更懂"的信念时，Gruber 指出这不是家长主义——这是设计。

**iPhone 经典案例**：2007 年，所有智能手机都有实体键盘。Jobs 在发布会上直接说"我们不这么做"。当时的智能手机用户说"我整天用手机打字，不可能用触摸屏"。结果——几年内，所有手机都变成了全触屏。

**核心区别**：设计师的工作不是"装饰"，而是决定产品应该"成为什么"。

# 软件质量的根源：CEO 的用户体验

**Gerstner 的历史类比**

"2000 年我们铺设光纤时，我们称之为暗光纤 (dark fiber) 是有原因的——没有人使用它。我们铺设时就知道这一点。"

**今天**："世界上没有一块闲置的 GPU。没有一个 token 是多余的。每个 token 都在被使用。"

**与互联网泡沫的根本区别**：
- 2000 年：3500 万人接入宽带。我们今天有 40 亿
- 2000 年：铺设光纤但无人使用。今天：每一块 GPU 都在服役，每一个 token 都有需求
- Google、Amazon、Microsoft、OpenAI、Anthropic **全部**报告 token 供给受限

</div>

<div>

<div class="p-4 rounded bg-blue-50 border-l-4 border-blue-500 text-sm leading-relaxed">

**供给侧的物理瓶颈**：
- 全球内存晶圆产能有限
- 全球逻辑晶圆产能有限
- 全球可用电力站址有限

→ 这意味着"我们能生产的 Token 数量有物理上限"

</div>

<div class="p-4 rounded bg-purple-50 border-l-4 border-purple-500 text-sm leading-relaxed mt-3">

**Gerstner 的总结**："是的，会有波动。但未来 9 个月，这些新模型产生的智能将让我们震惊。我们处于一个非常不同的时刻——不是因为泡沫，而是因为物理供给约束。"

</div>

# Trump 账户的"全民参与"设计

<div class="text-sm leading-relaxed mt-4">

<div class="grid grid-cols-3 gap-3">

<div class="p-3 rounded bg-purple-50 border-l-4 border-purple-500 text-sm">

529 计划服务于能存钱的美国前 10% 家庭。Trump 账户是**普惠的**——每一个孩子，无论家庭收入如何，自动获得。这**不是**政府项目——这是**私人账户**和**私人所有权**，可以终身复利。

- 75% 黑人和拉丁裔学校，服务农村贫困家庭
- "一位母亲哭着对我说：'我从没想过我的孩子会拥有任何东西'"

**2027 年起**：当年出生的 370 万孩子自动获得账户——"拿到社保号就拿到 Trump 账户"

</div>
