SpaceX：3 万亿美元市值的里程碑

当天最大的新闻来自 SpaceX。盘后交易中，SpaceX 市值突破 $3 万亿，轻松超越 Amazon，与 Alphabet 和 Apple（各约 $4 万亿）并驾齐驱，仅略低于 NVIDIA（约 $5 万亿）。这一数字令人瞠目——就在几个月前，人们还在猜测它能否触及 $2.4 万亿。

更引人注目的是 Cursor 的 600 亿美元并购案。这是 VC 支持创业公司中最大规模的并购，SpaceX 以新发行股票支付，在锁定期到期前用"新印的、低流通量的、零售通胀的货币"完成收购。正如一位观察者指出：SpaceX 的市值在交易后翻了两番以上，相当于"白送了"Cursor。

"这周我们经历了有史以来最大的 VC 支持 IPO 和最大的 VC 支持战略出售。一个年轻创业公司被以超过 500 亿美元收购——我们从未见过这种事。600 亿美元本身就已经远超 home run——而它发生在一笔并购中。" — TBPN 主持人

Cursor 的轨迹本身就是一个传奇。创始人 Michael Truell 多年不给自己发工资。公司一度占 Anthropic 收入的 40-50%——然后被云超大规模客户超越。当 Anthropic 最初告诉 Cursor "Claude Code 只是研究项目"时，没人预料到 Claude Code 后来会成为 Anthropic 不能没有的终端产品。

Tasklet：AI 智能体穿越围墙花园

Andrew Lee 是 Firebase 的联合创始人（2014 年卖给 Google），这是他第二次参加 YC。Tasklet 最初是一个"更好的 Gmail"项目——直到 LLM 变得足够强大，团队将 AI 智能体从邮件客户端中剥离出来，创建了一个全新的产品。

结果令人震撼：今年年初 run rate 约 $30 万，现在已到 $700 万。已完成 $2000 万 A 轮（USV、Lightspeed），正在融更多。Tasklet 直接与 Anthropic 和 OpenAI 竞争，而 Andrew 完全坦诚："是的，我们在它们的路径上。但我们可以执行得更好。"

"两年前如果你想集成很多东西，你必须手写一大堆集成。我们的秘密武器是 AI 可以动态生成集成——当你需要时当场创建。即使你要求一个我们不支持的集成，AI 也能为你生成一个……所以你可以用 API 连接这个，用 MCP 连接那个，用预构建连接器连接另一个——它们可以协同工作。" — Andrew Lee, Tasklet CEO

一个生动的案例：为准备 Demo Day，Andrew 让 Fable 5 模型接入他的 Gmail，搜索过去 10 个月内所有给他发冷邮件的人中谁会在 Demo Day 出现——生成一份排好优先级的 100+ 人电子表格。花了半小时、花了 $60 的推理费用。100% 准确，没有遗漏任何人。

Eden Robotics：$10/小时的机器人劳动力

Stamatios Floratos 以一种出人意料的方式进入机器人赛道。Eden Robotics 不做全人形机器人——而是做"半人形"：轮式底座加上两只在人肩高度的手臂，配备夹爪而非灵巧手。关键洞察：夹爪可以完成 80% 以上的工业任务，而腿式的能耗和成本高得离谱。

但真正颠覆的是商业模式。Eden 不卖机器人、也不按月出租——按使用小时计费，每小时 $10。客户拥有雇佣人类工人的思维框架，这让他们更容易接受。

"我们出售劳动力。我们甚至不按月租赁。我们按小时收费——$10/小时。我们发现客户比租赁概念更喜欢这个想法，因为他们没有租赁的思维框架……但他们有人类工人的思维框架。" — Stamatios Floratos, Eden Robotics CEO

目前使用汽车电池可运行 12 小时，自研电池设计可达到 20 小时。执行器寿命约 3 年——在 YC 校友 Demo Day 上连续运行 10 小时，无任何硬件故障。这是一个经过充分思考的经济模型：机器人不需要回家、不需要睡觉、不需要福利——但维护成本由 Eden 承担。

防御科技的创业浪潮

本期最鲜明的趋势之一是防御科技的爆发。YC 本批次的国防公司数量超过了去年全年总和——这不是偶然。Garry Tan 给出了一个直白的解释：

"Cost-Plus 七八十年前设计用来促进创新……五十年、八十年后，需要的东西已经完全不同了。国防巨头们实际上无法创新——这是结构性的。有服役人员在为保卫自由献出生命，因为国防巨头没有做好他们应该做的工作。不是个人的问题——是结构性的。" — Garry Tan, YC 总裁兼 CEO

Hugo Frisk 的 Tenet Industries 将电子制造和汽车工业的大规模可制造性理念带入国防领域。"比中国更便宜"是他们的口号。一个月内订单从 5 个飙升至 350 个。他的核心洞察：自主导航——而非光纤控制——才是解决无人机续航问题的终极答案。"末段瞄准在 60 年代就用导弹解决了。我搞不懂为什么有公司在推销这个。"

Russell Smith 的 9 Mothers 则从 YC Summer 2012（与 Coinbase、Instacart 同批）出发，现在为国防部打造 AI 反无人机系统——一个 35 磅重的机器人，售价 $15 万，可动能摧毁多个以 65 英里/小时飞行的无人机。公司已交付实弹系统，正开发全球首款弹链供弹霰弹枪。今年的目标是 100 台，明年 1000 台，预计明年营收超 $1 亿。

Adelante：$250 的全身 MRI

Efraín Torres 和他的联合创始人在研究生院开发的 MRI 技术，目标是将全身癌症筛查的价格打下来——从 $2500-3500（Prenuvo 的定价）降到 $250。系统轻 80%、功耗低 60%、安静 8 倍，设备寿命 10-12 年，毛利率 55%。

最聪明的市场策略：今年先用传统 FDA 批准的 MRI 装在福特 F-350 卡车上部署——建立分销渠道。已有 11 个意向书（$1975 万）。2028 年 Q1 自有产品获批后，直接在拖车里一换——"像一个利润杠杆"。Efraín 自己已经给自己扫描了上千次（MRI 没有致癌辐射，与 CT 和 X 光不同）。

Madrone：数据中心的电力密码

Akshay Trikha 指出了一个被大多数人忽视的真相：数据中心的瓶颈不是芯片、不是土地——是电力。冷却消耗高达 40% 的电力，而这些电力本可以分配给 GPU。在德州，去年只有 1.5 GW 的电网请求获批——而请求总量是 150 GW。

"数据中心的瓶颈不是芯片或土地——是电力。冷却和电力是同一枚硬币的两面。如果你能在冷却上省下电力，那就是更多给计算用的电力。人们可以在同样的电网许可证下安装更多 GPU——而这些许可证需要数年才能拿到。" — Akshay Trikha, Madrone 联合创始人

Madrone 的解决方案是一个全新的热力学过程，专为数据中心所在的干热气候优化，可减少电力和水消耗 30%。全栈自研——热力学、机械、电子、软件全部由两位创始人完成。"我们是 Tesla 的粉丝。所有竞争对手都把每一层工程外包，然后集成起来当产品卖。"100 kW 原型已在 Demo Day 现场运行，1 MW 版本即将部署。

YC 生态趋势：速度、学位与智能体经济学

Diana Hu 分享了本期最惊人的趋势数据：公司 3 个月内从零到七位数营收"已经比以前更普遍"。"当年我做 YC（2017 年暑期批次），最好的公司也只能做到每周 10% 的增长，到达几十万 ARR 就非常了不起了。"驱动变化的唯一因素：Agentic coding 让产品成熟速度发生了质的飞跃。

但同时，CS 学位正在大幅下降——Stanford 同比 -42%，Berkeley -61%。Harj Taggar 的解读是乐观的：聪明的人正在转向物理、数学和第一性原理学科，因为编程本身正在被 AI 工具覆盖。"以前我们说创造人们想要的东西——现在更多是创造智能体想要的东西。"

"当 Claude Code 面世时，情绪确实转向了'实验室会拿走更多价值'。但现在它又摆回来了——创业者们在想的是：那些无聊但有价值的东西，大实验室肯定不会去做。' — Harj Taggar, YC 管理合伙人

Harj 还分享了一个 YC 内部关于 Token 支出的态度转变：YC 合伙人认为 Token 应该像租金——"你不会因为旧金山租金贵就不搬来建你的科技公司。该花钱的地方就是这里。省别的东西。"但对大多数年轻创始人来说，他们习惯了节约，订阅计划目前还很慷慨——这个转变还需要时间。

Garry Tan：煮沸海洋

Garry Tan 带来了 YC 的灵魂宣言。他从结构性问题讲起——大科技的官僚病（"Siri 有多蠢"）、国防巨头的不作为——然后提升到一个更宏大的命题：现在是"煮沸海洋"的时代。

"世界到处都是问题——如果我们能加速技术来解决它们……阻碍不是钱，是很久以前就不是了。是纯粹的主动性：你有没有想法？你能用提示词实现它吗？你想在这个世界上创造什么？" — Garry Tan

他分享了 G-Brain 项目——一个"智能体的 Postgres"。人类大脑只能记住约 7±3 件事（一个电话号码的长度），LLM 可以记住约 3 本哈利波特的内容。但公司拥有的是整个亚历山大图书馆——数百万个 Markdown 文件。G-Brain 要做的是让智能体加载"恰好需要的 3 本书"来完成任务。

PG 的新训诫也被 Garry 引用："如果你是非技术背景——不要再叫自己非技术背景。坐在 Claude Code 面前，你可以修 Python bug、提交会被接受的 PR。一年前这还不成立。"

Threads：5 亿月活的"安静应用"

Connor Hayes 带着里程碑数字出场：Threads 正式突破 5 亿月活用户。但真正值得注意的是增长的质量——来自非 Facebook/Instagram 推广的自然流量比例持续上升。日本的参与时间同比增长 130%，韩国增长 80%。

Threads 的用户称它为"安静应用"——在每款应用都是自动播放视频的时代，这里有文字、有对话、没有人逼你看不该看的东西。Connor 坚定地拒绝了竖屏视频标签页的诱惑：

"我知道我可以明天就靠加视频来提升使用时长——但那不会让我们成为最好的公共对话应用。所以不会做。Threads 不会有竖屏视频标签页。" — Connor Hayes, Threads 负责人

Connor 的策略是市场聚焦而非全球撒网——在一个一个国家里把 Threads 做成"公共对话的默认应用"。目前东南亚已经初见成效，美国是下一个重心。成立不到 3 年的 Threads，仍然被认为是 Meta 生态系统中的"年轻应用"。

Luke Burgis：AI 时代的模仿欲望

Luke Burgis 带着新书《The One and the 99》来到节目，开场就恭喜两位主持人："你们在媒体历史上为制造模仿欲望做了最多的工作——谈论谁在融资、谁在退出、规模有多大……还有那个锣。"

他的核心论点令人不安：AI 可能正在制造一种我们尚未完全理解的传染效应。当 LLM 总结 Tim Ferriss 的书并高度个性化地呈现给你——却删除了"Tim Ferriss"这个名字——你感觉这些洞见是为你定制的，而实际上你正在做着和成千上万问了同样问题的人完全一样的事。

"当你 max 任何一件事的时候，你在放弃什么？你不能 max 一切。我在二十多岁的时候就是这样——max 一间公司，甚至都不喜欢它 → 腻了 → 再 max 另一间。" — Luke Burgis

他对"坚固的自我"与"伪自我"的区分尤为深刻。伪自我进入不同房间就随环境调整；坚固的自我即使面对群体压力也能保持内在一致性。而他的解决方案出人意料：学会休闲。"这是一种被低估的技能——我记得在泰国海滩上为初创公司焦虑，完全无法静坐。我们现在在 Clooney Institute 为创始人提供人文静修。"

Anda Gansca：网站已死，知识站永生

最后一棒的 Anda Gansca（Notch CEO）带来了一个已经发生的现实：智能体网络流量已经超过了人类流量（Matthew Prince / Cloudflare 的分享）。

品牌们在恐慌中开始优化网站让智能体可读——结果却是大量低质量内容挤满了本来为人设计的页面，导致人类的即时跳出率飙升。"LLM 不能被游戏化。GEO 的那些伎俩——再发 100 篇博客、10 个 FAQ 页面——最终彻底毁了人类体验。"

Notch 的 ACE 平台给出的答案是：为品牌知识建立一个单一的向量化和语义标记的知识库，然后根据来访者的身份（人类还是智能体）以及输入的语义意图，动态地重组页面。同一个知识库，两种呈现形态。人类看到的是个性化网站——智能体看到的是结构化代码。

"网站，就我们目前所知，肯定是死了。我们要用'知识站'来替代这个词。在明天的世界里——无论人类是否继续访问你的站点——重要的是你的知识是否存在于一个可以与智能体对话的知识库中。" — Anda Gansca, Notch 联合创始人兼 CEO

Notch 自己的网站在 Demo Day 前 48 小时内用 ACE 重建。Anda 的野心不限于 Fortune 500："我们要覆盖所有地方。人类不会消失——每出现一个新技术趋势，我们都听到'这就是终点'的说法。我不相信零点击营销。我相信选择。"
