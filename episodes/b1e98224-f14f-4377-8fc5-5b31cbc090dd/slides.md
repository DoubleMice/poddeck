# 模型编号的通货膨胀

<div class="text-sm leading-relaxed">

**现状**：模型编号已经失去原有含义。以前编号 = 预训练版本 + 后训练版本，现在各家随意跳号。

- GPT-5.6 → Google 3.5 Pro → Muse Spark 1.1 → Grok 4.5
- 编号越来越像汽车型号：BMW 5 系 vs 2027 款
- 社区预测：年底前会出现编号"6"的模型

**本质**：后推理时代（post-reasoning），模型有两种不同的扩展路径——预训练和后训练。一个数字已经不够用了。

</div>

# Meta 广告帝国的 AI 飞轮

<div class="text-sm leading-relaxed">

Eric Seufert 的核心论点：

**为什么图像模型对 Meta 特别重要？**
广告主最大的瓶颈是"缺少好创意"。Meta 拥有独特的闭环数据——从广告曝光到转化（ROAS），通过 Pixel 和 CAPI 回传。这是**只有 Meta 才有的训练数据**。

**飞轮效应**：
1. 平台拥有广告效果数据
2. 用这些数据训练图像/视频模型
3. 模型生成更高效的广告创意
4. 更好的广告 → 更多收入 → 更多数据

</div>

# 仿生手的设计哲学

<div class="text-sm leading-relaxed">

Bernt 详细阐述了三个核心设计原则：

**1. 安全性 = 学习能力**
软性柔性系统使力可以双向流动——机器人既能感知世界，也能作用于世界，手指不会在探索中损坏。

**2. 与人类行为对齐**
高度模仿人手的非线性柔性意味着：互联网上所有人类操作视频都可以成为训练数据——这大大降低了机器人智能的数据门槛。

**3. 最简复杂性**
手很复杂，但它是"能完成人类手部任务的最简方案"。不使用齿轮或快速移动部件——一切遵循自然肌肉运动逻辑。
